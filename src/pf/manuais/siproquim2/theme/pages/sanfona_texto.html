    1. Acesse a aba Dados do Requerente para inserir os dados de identificação da pessoa jurídica requerente.
    2. Caso seja necessário que adicione algum anexo siga o item Anexos logo abaixo.
    3. Caso não, clique em Salvar para que os dados sejam salvos e as outras abas sejam habilitadas.

    Dica: Para solicitar o requerimento do tipo CRC, é necessário ter o E-CNPJ do requerente conectado ao computador para que seja feita a leitura e a validação. Logo após esse processo, alguns dados básicos do requerente serão recuperados da base da Receita Federal.

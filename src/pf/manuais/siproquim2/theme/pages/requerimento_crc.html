Requerimento CRC

O Certificado de Registro Cadastral - CRC, tem por finalidade o registro quando se trata de uma matriz que não exerce atividades com produtos químicos controlados.

Ao acionar a opção CRC, o sistema apresenta a tela com o filtro de pesquisa e a lista de requerimentos do tipo CRC. A partir do resultado da pesquisa você poderá alterar ou excluir um requerimento, ou solicitar um novo requerimento cadastral.

Novo requerimento cadastral CRC

Para solicitar um novo requerimento cadastral do tipo CRC, clique em Novo Requerimento Cadastral. Será apresentada a aba Início, com informações sobre o tipo de requerimento emitido.

  • Dados do requerente navigate_next

    1
    Acesse a aba Dados do Requerente para inserir os dados de identificação da pessoa jurídica requerente.

    2

    Caso seja necessário que adicione algum anexo siga o item Anexos logo abaixo.

    3

    Caso não, clique em Salvar para que os dados sejam salvos e as outras abas sejam habilitadas.

    Dica: Para solicitar o requerimento do tipo CRC, é necessário ter o E-CNPJ do requerente conectado ao computador para que seja feita a leitura e a validação. Logo após esse processo, alguns dados básicos do requerente serão recuperados da base da Receita Federal.

Alterar requerimento CRC

Para alterar um requerimento cadastral do tipo CRC:

1

Acesse no menu lateral o item Emissão e logo após CRC.

2

Acesse o requerimento que deseja alterar na lista de Requerimentos de CRC, o link fica da coluna "Requerente".

3

As informações do requerimento serão recuperadas e será apresentada a aba Início.

Você poderá alterar as informações enquanto o requerimento estiver com status igual a “Salvo”; no status “Pendente”, somente os campos que contenham pendências serão habilitados para alteração, os campos desabilitados ficam em cinza. Para atualizar os campos habilitados, você pode seguir os passos mostrados anteriormente no guia em Requerimento CRC.

Dica: O ícone de pendência 🛑 só é apresentado para requerimentos com status igual a "Pendente" nos campos em que o analista registrou uma divergência de dados. Ao clicar, será aberta uma modal contendo a descrição da pendência.
